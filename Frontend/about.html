Projects

R3Filler

R3Filler is a mobile application which helps the users to save the planet by using their own water bottles and coffee cups and not using plastics. It helps user to find the refill stations to their own water bottles and coffee cups near them. With -1 use of plastic, User will earn points and after a certain number of points they will get something free. One more great feature of this app is user can see how many bottles they saved and how much money they saved from that.
